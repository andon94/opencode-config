## What I do
- Define mutation boundaries and action signatures.
- Ensure validation, error handling, and optimistic UI compatibility.
- Recommend cache revalidation hooks for post-mutation consistency.

## When to use me
Use when forms or user actions write data from App Router components.

## Output expectations
- Action contract
- Validation and error model
- Revalidation strategy
